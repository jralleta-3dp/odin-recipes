# odin-recipes
A repository that holds the first project in the foundations section of The Odin Project's curriculum.

Current project:

Recipe Book

This is a website that consists of multiple pages. Where each page is a different recipe that I would like
to cook at home! The purpose of this project is to hone and sharpen the HTMl skills that I have learned thus
far into The Odin Project's curriculum. 

The website will consist of a main index.html page, from which the user will be able to navigate to 
ohter web pages containing recipies using relative links.

- Heading structure
